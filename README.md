# Defuser MVP

Браузерный прототип кооперативной игры `Сапер и Координатор`.

## Что уже есть

- `React + Vite + Tailwind` сайт с лобби, кодом комнаты, ссылкой-приглашением и экраном ролей.
- `Fastify` API для создания комнаты, входа по коду, настройки раунда и старта матча.
- `Socket.io` для обновления состояния комнаты в реальном времени.
- `SQLite` через встроенный `node:sqlite` для сохранения комнат и событий.
- Локальная база хранится в `apps/server/storage/defuser.sqlite`.
- Несколько типов модулей: провода, кнопка, тумблеры, символы, лампочки и диск.

## Запуск

1. Установить зависимости:

```bash
npm install
```

2. Создать `.env` на основе `.env.example`.

3. Для быстрого старта запустить весь стек:

```bash
npm run dev
```

4. Если нужно запускать раздельно:

```bash
npm run dev:server
npm run dev:web
```

## Локальные адреса

- frontend: `http://localhost:5173`
- backend: `http://localhost:3000`
- health backend: `http://localhost:3000/api/health`

## Проверка

```bash
npm run check
npm run build
```

## Как играть в прототип

1. Первый игрок создает комнату на сайте.
2. Он копирует ссылку или передает код комнаты напарнику.
3. Второй игрок открывает ссылку или входит по коду.
4. Владелец комнаты настраивает время, количество и типы модулей.
5. После старта роли распределяются случайно.
6. `Сапер` видит панель модулей, `Координатор` получает только инструкции.
